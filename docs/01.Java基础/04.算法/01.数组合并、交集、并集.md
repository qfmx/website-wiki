# 数组合并、交集、并集
## 数组合并
```java
public static int[] merge(int[] a, int[] b) {
    int[] c = new int[a.length + b.length];
    int i = 0, j = 0, k = 0;
    while (i < a.length && j < b.length) {
        if (a[i] < b[j]) {
            c[k++] = a[i++];
        } else {
            c[k++] = b[j++];
        }
    }
    while (i < a.length) {
        c[k++] = a[i++];
    }
    while (j < b.length) {
        c[k++] = b[j++];
    }
    return c;
}
```
## 数组的交集
```java
public static int[] intersection(int[] a, int[] b) {
    int[] c = new int[a.length];
    int i = 0, j = 0, k = 0;
    while (i < a.length && j < b.length) {
        if (a[i] < b[j]) {
            i++;
        } else if (a[i] > b[j]) {
            j++;
        } else {
            c[k++] = a[i++];
            j++;
        }
    }
    return Arrays.copyOf(c, k);
}
```

## 数组的差集
```java
public static int[] difference(int[] a, int[] b) {
    int[] c = new int[a.length];
    int i = 0, j = 0, k = 0;
    while (i < a.length && j < b.length) {
        if (a[i] < b[j]) {
            c[k++] = a[i++];
        } else if (a[i] > b[j]) {
            j++;
        } else {
            i++;
            j++;
        }
    }
    while (i < a.length) {
        c[k++] = a[i++];
    }
    return Arrays.copyOf(c, k);
}
```
